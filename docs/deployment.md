# Deployment

TrustTill is currently a static prototype. It can be deployed anywhere that serves static HTML.

## Fastest Local Run

Open `index.html` directly in a browser.

## GitHub Pages

1. Create a GitHub repository named `trusttill-hackaholics`.
2. Upload all files in this folder.
3. Go to repository **Settings**.
4. Open **Pages**.
5. Set source to the main branch and root folder.
6. Use the generated GitHub Pages URL as the portfolio/demo link.

## Vercel

1. Create a GitHub repository named `trusttill-hackaholics`.
2. Push this folder.
3. Import the repository into Vercel.
4. Choose static project defaults.
5. Deploy.

## Netlify

1. Drag this folder into Netlify Drop, or connect the GitHub repo.
2. Use the generated URL as the demo URL.

## Submission Tip

For the Hackaholics application field called **Portfolio/GitHub**, use the public GitHub repository link if the demo is not hosted yet. If hosted, use the live demo link and put the GitHub repo in the README.
